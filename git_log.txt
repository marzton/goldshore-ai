    - **Reorganized and documented** all tokens with section headers for

    - **Updated color values**: primary background (#0b0f18), accent blue

    - **Added comprehensive spacing scale** (--gs-space-1 through

    - **Added derived tokens**: soft accent, contrast, faint colors, shadows
    (soft/hard), and transitions
    - **Added layout helpers**: section padding/gap, grid minimum widths,
    max-width tokens
    - **Added app-level shorthands** (--gs-bg, --gs-text, --gs-primary,
    etc.) for convenience
    - **Removed duplicate definitions** from
    `apps/gs-web/src/styles/global.css` (now sourced from canonical tokens)
    
    ### Preview Deployment Infrastructure
    - **Added preview DNS setup script** (`scripts/setup-preview-dns.mjs`):
    One-shot Cloudflare API automation for CNAME and custom domain
    configuration
    - **Updated Cloudflare desired state**
    (`infra/Cloudflare/desired-state.yaml`):
    - Added preview CNAME record (preview.goldshore.ai →
    preview-web.pages.dev)
      - Added preview-web Pages project configuration
      - Fixed package filter names (@goldshore/gs-web, @goldshore/gs-admin)
      - Added gs-agent Worker with queue bindings
      - Updated email forwarding placeholders with TODO comments
    - **Updated GitHub Actions workflows** (preview-gs-web.yml,
    preview-gs-admin.yml):
      - Added push trigger for claude/* branches with path filters
    - Dynamic release_label generation (pr-N for PRs, branch-name for
    pushes)
      - Conditional job execution for both push and PR events
    
    ### Accessibility & Motion Improvements
    - **Enhanced focus handling** (`packages/theme/src/styles/base.css`):
    - Changed input focus from `:focus` to `:focus-visible` (keyboard-only)
      - Added global `:focus-visible` ring with 2px outline and 3px offset
    - **Improved prefers-reduced-motion**
    (`packages/theme/src/styles/motion.css`):
      - Added html scroll-behavior reset
    - Extended reduced-motion rules to cover view-timeline and 3d-tilt
    animations
    
    ### Configuration & Documentation
    - **Added Claude Code project config** (`.claude/README.md`):
    Instructions for local Cloudflare credentials setup
    - **Updated .gitignore**: Added .claude/settings.local.json to prevent
    secret leakage
    - **Updated Wrangler compatibility dates**: gs-agent and gs-api now
    target 2025-01-10
    
    ## Notable Implementation Details
    - Design tokens now follow a hierarchical structure with semantic
    grouping and clear section boundaries
    - Preview deployments support both PR and branch-based workflows via
    dynamic release labeling
    - Accessibility improvements prioritize keyboard navigation without
    affecting mouse users
    - All color and spacing values are now centralized in the canonical
    tokens file, eliminating duplication
