# Agent Instructions

This file defines repository-wide default instructions for AI agents.

For precedence, scoped overrides, and validation workflow, see `docs/ops/instruction-governance.md`.

## Tagging for Review

To request a review of an error or issue, use one of these tags in comments or pull request descriptions:

- **@Jules-Bot `[review-request]`**: General code review.
- **@Jules-Bot `[error-analysis]`**: Diagnose and fix a specific error.
- **@Jules-Bot `[issue-repro]`**: Reproduce a reported issue.

When using a review tag, include context:

- A clear description of the issue or code being reviewed.
- Reproduction steps (for bugs/issues).
- Relevant logs or error messages.
- Expected outcome.
