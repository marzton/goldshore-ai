# Agent Instructions

This file defines repository-wide default instructions for AI agents.

For precedence, scoped overrides, and validation workflow, see `docs/ops/instruction-governance.md`.

## Tagging for Review

To request a review of an error or issue, use one of these tags in comments or pull request descriptions:

- **@Jules-Bot `[review-request]`**: For a general code review.
- **@Jules-Bot `[error-analysis]`**: For help in diagnosing and fixing a specific error.
- **@Jules-Bot `[issue-repro]`**: For assistance in reproducing a reported issue.

When using a review tag, include context:

- A clear description of the issue or the code to be reviewed.
- Steps to reproduce the error or issue.
- Any relevant logs or error messages.
- The expected outcome.
